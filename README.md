# Hand Gesture Recognition with Artificial Neural Network

This project is a hand gesture recognition system developed to convert hand movements into computer commands using Artificial Neural Networks (ANN).

The model was trained on a custom dataset created manually for this project. Development and experimentation were carried out in Jupyter Notebook.

## Features

- Hand gesture detection and recognition
- Custom-built dataset
- Artificial Neural Network (ANN) model
- Gesture-to-command conversion
- Developed and tested in Jupyter Notebook

## Technologies Used

- Python
- Jupyter Notebook
- TensorFlow / Keras
- NumPy
- OpenCV
- Scikit-learn

## Dataset

The dataset used in this project was created manually and contains different hand gesture samples collected for training and testing purposes.

## Project Goal

The main objective of this project is to enable interaction with a computer through hand gestures by translating recognized movements into executable computer commands.
